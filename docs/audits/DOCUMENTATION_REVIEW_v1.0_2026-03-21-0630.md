## Naming Convention Standard

### Adopted Format
```
{DOCUMENT-NAME}_v{VERSION}_{YYYY-MM-DD-HHMM}.md
```

### Examples
- `PRODUCT-BRIEF_v1.0_2026-03-21-0630.md`
- `ADR-001-MULTI-TENANCY_v1.0_2026-03-21-0500.md`
- `STAKEHOLDER-AUDIT_v1.0_2026-03-21-0630.md`

### Versioning Rules
- **v1.0:** Initial release
- **v1.1:** Minor updates, corrections
- **v2.0:** Major revision, restructuring
- **Date:** Creation date (not modification date)
- **Time:** 24-hour format UTC
